9. Словари

Словарь — неупорядоченная структура данных, которая позволяет хранить пары «ключ — значение».

Определение словаря имеет следующий синтаксис:

dictionary = { ключ1:значение1, ключ2:значение2, ....}

В фигурных скобках через запятую определяется последовательность элементов, где для каждого элемента сначала указывается ключ и через двоеточие его значение.

Также для получения элементов можно использовать метод get, который имеет две формы:

  • get(key): возвращает из словаря элемент с ключом key. Если элемента с таким ключом нет, то возвращает значение None
  • get(key, default): возвращает из словаря элемент с ключом key. Если элемента с таким ключом нет, то возвращает значение по умолчанию default

    Другой способ удаления представляет метод pop(). Он имеет две формы:

  • pop(key): удаляет элемент по ключу key и возвращает удаленный элемент. Если элемент с данным ключом отсутствует, то генерируется исключение KeyError
  • pop(key, default): удаляет элемент по ключу key и возвращает удаленный элемент. Если элемент с данным ключом отсутствует, то возвращается значение default

    • Если необходимо удалить все элементы, то в этом случае можно воспользоваться методом clear():

    users.clear()

    Задача 1: Фрукты

    Условие: Пользователь вводит число K - количество фруктов. Затем он вводит K фруктов в формате:

    название фрукта и его количество. Добавьте все фрукты в словарь,

    где название фрукта - это ключ, а количество - значение. Например:

    Показать ответ

    # Запрашиваем у пользователя количество фруктов

    K = int(input("Введите количество фруктов: "))

        

    # Создаем пустой словарь для хранения фруктов

    fruits_dict = {}

        

    # Цикл для ввода фруктов

    for i in range(K):

        # Запрашиваем название фрукта

        fruit_name = input(f"Введите название фрукта {i + 1}: ")

        # Запрашиваем количество этого фрукта

        fruit_quantity = int(input(f"Введите количество {fruit_name}: "))

        

        # Добавляем фрукт в словарь

        fruits_dict[fruit_name] = fruit_quantity

        

    # Выводим итоговый словарь с фруктами

    print("Итоговый словарь фруктов:")

    print(fruits_dict)

    Задача 2: Старший и младший

    Пользователь вводит число N. Затем он вводит личные данные (имя и возраст) своих N друзей.

    Создайте список friends и добавьте в него N словарей с ключами name и age.

    Найдите самого младшего из друзей и выведите его имя.

    Показать ответ

    friends = []

    ages = [] #создадим список, в котором будем хранить все возраста

    names = []#создадим список, в котором будем хранить все имена

    n = int(input())

    for r in range(n): #наполняем список необходимым количеством словарей

        friends.append({})

        

    for j in range(n): #наполняем словари

        name = input()

        age= int(input())

        friends[j][name] = age

        ages.append(age) #наполняем списки

        names.append(name)

        

    friend = ages.index(min(ages)) #ищем самого младшего через списки имен и возрастов

    print("Самый младший друг:", names[friend])

    Создайте список friends и добавьте в него N словарей с ключами name и age.

    Выведите средний возраст всех друзей и самое длинное имя.

    Показать ответ

    friends = []

    n = int(input())

    for r in range(n): #наполняем список необходимым количеством словарей

        friends.append({})

        

    for j in range(n): #наполняем словари

        name = input()

        friends[j][name] = age

        ages.append(age) #наполняем списки

    Задача 4: Английский словарь

    Программа получает на вход слово на английском языке и несколько его переводов на русском языке.

    Составьте словарь, в котором ключ - это английское слово, а значение - это список русских слов.

    В этой задаче нужно использовать строчный метод split().

    Показать ответ

    # Спрашиваем у пользователя английское слово

    english_word = input("Введите английское слово: ")

            

    # Спрашиваем переводы на русском языке

    translations_input = input("Введите переводы на русском языке через запятую: ")

            

    # Разделяем переводы по запятой и убираем лишние пробелы

    translations_list = translations_input.split(',')

            

    # Убираем пробелы у каждого перевода

    for i in range(len(translations_list)):

            translations_list[i] = translations_list[i].strip()

            

    # Создаем словарь

    dictionary = {english_word: translations_list}

            

    # Показываем результат

    print(f"Ваш словарь: {dictionary}")

    Задача 5: Магазин

    Создайте словарь с названиями продуктов и их ценами.

    Напишите программу, которая позволяет пользователю вводить

    количество каждого продукта и вычислять общую стоимость.

    (список наполните сами)

    Показать ответ

    products = {

        "яблоко": 100,

        "банан": 80,

        "апельсин": 90

    }

        

    total_cost = 0

        

    for product, price in products.items():

        quantity = int(input(f"Введите количество {product}: "))

        total_cost += price * quantity

        

    print(f"Общая стоимость: {total_cost} руб.")
