6. Цикл while

Цикл while ("пока") позволяет выполнить одну и ту же последовательность действий, пока проверяемое условие истинно. Условие записывается после ключевого слова while и проверяется до выполнения тела цикла.

Цикл while используется, когда невозможно определить точное количество повторений цикла.

Пример простого цикла, который выводит от 0 до 4

Задачи

Задача 1: Сумма чисел

Условие: Напишите программу, которая будет запрашивать у пользователя целые числа,

пока он не введет 0. Программа должна выводить сумму всех введенных чисел (кроме 0).

Показать ответ

total = 0

number = int(input("Введите целое число (0 для выхода): "))

while number != 0:

    total += number

    number = int(input("Введите целое число (0 для выхода): "))

print("Сумма введенных чисел:", total)

Задача 2: Подсчет четных и нечетных чисел

Условие: Напишите программу, которая запрашивает у пользователя целые числа до тех пор,

пока он не введет 0. Программа должна подсчитать и

вывести количество четных и нечетных чисел.

Показать ответ

even_count = 0

odd_count = 0

number = int(input("Введите целое число (0 для выхода): "))

while number != 0:

    if number % 2 == 0:

        even_count += 1

    else:

        odd_count += 1

    number = int(input("Введите целое число (0 для выхода): "))

print("Количество четных чисел:", even_count)

print("Количество нечетных чисел:", odd_count)

Задача 3: Обратный отсчет

Условие: Напишите программу, которая делает обратный отсчет

от заданного пользователем числа до 0.

Показать ответ

n = int(input("Введите число для обратного отсчета: "))

while n >= 0:

    print(n)

    n -= 1

Задача 4: Проверка на простое число

Условие: Напишите программу, которая проверяет, является ли введенное пользователем

число простым. Используйте цикл while.

Показать ответ

n = int(input("Введите целое число для проверки на простое: "))

if n < 2:

    print(n, "не является простым числом.")

else:

    is_prime = True

    i = 2

    while i * i <= n:

        if n % i == 0:

            is_prime = False

            break

        i += 1

    if is_prime:

        print(n, "является простым числом.")

    else:

        print(n, "не является простым числом.")

Задача 5: Факториал

Условие: Напишите программу, которая вычисляет факториал числа,

введенного пользователем. Используйте цикл while.

Показать ответ

n = int(input("Введите целое число для вычисления факториала: "))

factorial = 1

i = 1

while i <= n:

    factorial *= i

    i += 1

print("Факториал", n, "равен", factorial)
