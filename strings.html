5. Строки

Строковый тип данных — это последовательности символов Unicode любой длины, заключённые в одинарные, двойные или тройные кавычки. Символами могут быть буквы, числа, знаки препинания, специальные символы и так далее. Главное — чтобы их с двух сторон окружали кавычки. В Python текстовый тип данных обозначается буквами str (от англ. string — строка).

Задачи

Задача 1: Обратная строка

Условие: Напишите программу, которая принимает строку и выводит её в обратном порядке.

Пример ввода: "Привет"

Пример вывода: "тевирП"

Показать ответ

s = input("Введите строку: ")

print(s[::-1])

Задача 2: Замена символов

Пример ввода: "Привет мир"

Пример вывода: "Привет_мир"

Показать ответ

s = input("Введите строку: ")

print(s.replace(" ", "_"))

Задача 3: Подсчет слов

Условие: Напишите программу, которая подсчитывает количество слов в строке.

Пример ввода: "Я люблю программирование"

Пример вывода: 3

Показать ответ

s = input("Введите строку: ")

word_count = len(s.split())

print(word_count)

Задача 4: Повтор строки

Условие: Напишите программу, которая повторяет строку заданное количество раз.

Пример ввода: "Привет" (количество: 3)

Пример вывода: "ПриветПриветПривет"

Показать ответ

s = input("Введите строку: ")

n = int(input("Введите количество повторений: "))

print(s * n)

Задача 5: Удаление всех вхождений символа

Условие: Напишите программу, которая удаляет все вхождения заданного символа из строки.

Пример ввода: "программирование" (символ: "р")

Пример вывода: "огаминирование"

Показать ответ

s = input("Введите строку: ")

char_to_remove = input("Введите символ для удаления: ")

print(s.replace(char_to_remove, ""))

Задача 6: Проверка наличия подстроки

Условие: Напишите программу, которая проверяет, содержится ли подстрока в строке.

Пример ввода: "программирование" (подстрока: "грам")

Пример вывода: "Да"

Показать ответ

s = input("Введите строку: ")

substring = input("Введите подстроку для поиска: ")

if substring in s:

    print("Да")

else:

    print("Нет")
