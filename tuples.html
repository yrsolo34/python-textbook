10. Кортежи

Кортеж (tuple) представляет последовательность элементов, которая во многом похожа на список за тем исключением, что кортеж является неизменяемым (immutable) типом. Поэтому мы не можем добавлять или удалять элементы в кортеже, изменять его.

Если вдруг кортеж состоит из одного элемента, то после единственного элемента кортежа необходимо поставить запятую:

tom = ("Tom",)

Задача 2: Студенты

Условие: Создайте кортеж с именами студентов в классе.

Показать ответ

Задача 3: Страны

Условие: Напишите программу, которая создает кортеж с тремя странами и их населением.

Выведите информацию о каждой стране.

Показать ответ

countries = (("Россия", 146599183), ("США", 331002651), ("Китай", 1439323776),)

for country, population in countries:

print(f"{country}: население {population}")

Задача 4: Зоопарк

Напишите программу, которая создает кортеж с именами животных.

Выведите первое и последнее имя из кортежа.

Показать ответ

animals = ("кот", "собака", "попугай", "кролик")

print(f"Первое животное: {animals[0]}")

print(f"Последнее животное: {animals[-1]}")

Задача 5: Кинотеатр

Условие: Напишите программу, которая создает кортеж с названиями фильмов.

Позвольте пользователю вводить номер фильма и выводите его название.

Показать ответ

movies = ("Фильм1", "Фильм2", "Фильм3", "Фильм4",)

index = int(input("Введите номер фильма (1-4): ")) - 1

if 0 <= index < len(movies):

    print(f"Выбранный фильм: {movies[index]}")

else:

    print("Неверный номер фильма.")
