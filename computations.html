Задачи

Задача 1: «Последняя цифра числа»

Условие: Дано натуральное число. Выведите его последнюю цифру.

Показать ответ

n = int(input("Введите число: "))

print(n % 10)

Задача 2: «МКАД»

Условие: Длина Московской кольцевой автомобильной дороги —109 километров. Байкер Вася стартует с нулевого километра МКАД и едет со скоростью v километров в час.

На какой отметке он остановится через t часов?

Программа получает на вход значение v и t. Если v>0, то Вася движется в положительном направлении по МКАД, если же значение v<0, то в отрицательном.

Программа должна вывести целое число от 0 до 108 — номер отметки, на которой остановится Вася.

Показать ответ

a = int(input())

b = int(input())

print((a * b) % 109)

Задача 3: «Гипотенуза»

Условие: Дано два числа a и b. Выведите гипотенузу треугольника с заданными катетами.

Показать ответ

a = int(input())

b = int(input())

c = (a**2 + b**2) ** 0.5 #получить корень - возвести число в 0.5 стпень

print(c)

Задача 4: «Сумма цифр»

Условие: Дано трехзначное число. Найдите сумму его цифр.

Показать ответ

n = int(input("Введите трехзначное число: "))

d1 = n % 10 # извлекается последняя цифра (младший разряд)

d2 = n % 100 // 10 #средняя цифра

d3 = n // 100 #первая цифра (старший разряд)

print("Сумма цифр числа:", d1 + d2 + d3)

Показать ответ

a = int(input())

b = int(input())
